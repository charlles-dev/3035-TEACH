# Tarefa 06 - Pokédex (Styled Components)

Aplicação que consome a PokeAPI para exibir cards de Pokémon, estilizada inteiramente com Styled Components.

## Funcionalidades
- Consumo de API externa (pokeapi.co).
- Grid responsivo de cards.
- Estilização avançada com Styled Components (Scoped CSS).
- Efeitos de hover e animações de entrada.

## Como Executar
1. Entre na pasta: `cd tarefa-06`
2. Instale: `npm install`
3. Execute: `npm run dev`
