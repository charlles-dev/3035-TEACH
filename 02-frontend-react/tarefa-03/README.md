# Tarefa 03 - Seleção de Filmes (Checkbox)

Esta aplicação permite selecionar filmes de uma lista utilizando checkboxes personalizados.

## Funcionalidades
- Lista de filmes com estilo de cartões.
- Checkboxes customizados (design CSS).
- Alerta visual ao selecionar um filme.

## Como Executar
1. Entre na pasta: `cd tarefa-03`
2. Instale: `npm install`
3. Execute: `npm run dev`
