# Tarefa 04 - Lista de Tarefas (Context API)

Uma aplicação "To-Do List" completa utilizando React Context API para gerenciamento de estado.

## Funcionalidades
- Adicionar novas tarefas.
- Marcar tarefas como concluídas (efeito riscado).
- Excluir tarefas.
- Interface minimalista e animada.

## Como Executar
1. Entre na pasta: `cd tarefa-04`
2. Instale: `npm install`
3. Execute: `npm run dev`
