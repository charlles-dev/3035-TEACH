# Tarefa 01 - Contador Interativo

Esta aplicação é um contador simples desenvolvido com React.

## Funcionalidades
- Exibe o número de cliques.
- Botão para incrementar o contador.
- Interface estilo "Glassmorphism" (efeito de vidro).

## Como Executar
1. Certifique-se de estar na pasta do projeto: `cd tarefa-01`
2. Instale as dependências (se não tiver feito): `npm install`
3. Inicie o servidor de desenvolvimento: `npm run dev`
