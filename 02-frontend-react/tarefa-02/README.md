# Tarefa 02 - Lista de Filmes com Filtro

Esta aplicação exibe uma lista de filmes em formato de tabela e permite filtrar por nome.

## Funcionalidades
- Tabela moderna e responsiva.
- Campo de pesquisa para filtrar filmes em tempo real.
- Design centralizado em cartão.

## Como Executar
1. Entre na pasta: `cd tarefa-02`
2. Instale: `npm install`
3. Execute: `npm run dev`
