# TeachGram Pro Frontend

Frontend React + TypeScript do desafio final.

## Stack

- React.
- TypeScript.
- Vite.
- React Router.
- TanStack Query.
- React Hook Form.
- Zod.
- CSS com tokens visuais extraidos do Figma.

## Como Rodar

```bash
cd frontend
npm install
npm run dev
```

URL local:

```text
http://localhost:5173
```
