# Tarefa 05 - Alternador de Tema (Context API)

Demonstração de mudança de tema (Claro/Escuro) utilizando Context API.

## Funcionalidades
- Botão para alternar entre modo Claro e Escuro.
- Transições suaves de cores (background, texto, sombras).
- Persistência de tema durante a navegação (via Contexto).

## Como Executar
1. Entre na pasta: `cd tarefa-05`
2. Instale: `npm install`
3. Execute: `npm run dev`
